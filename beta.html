Barcode Scanner Benchmark

EAN-13/ISBN 用に最適化
計測: 起動時間（カメラ起動クリック→ライブラリ初期化完了）と、認識開始→最初のバーコード検出までの時間を記録します。
注: 教科書ISBNは多くが EAN-13（978/979 プレフィックス）です。ZXing は EAN-13 のみに絞って高速化しています。
トラブルシュート:
  • HTTPS で配信してください（カメラはセキュアコンテキストが必要）。
  • iOS Safari ではユーザ操作後のみ再生可能。ボタンから開始してください。
  • 他タブや他アプリがカメラを使用中だと取得に失敗します。

このページの目的と設計

1つのHTML内で Quagga / Quagga2 / ZXing を切替え、(1)カメラ初期化の起動時間 と (2)「認識開始」→最初の検出までの時間 を比較計測します。 グローバル衝突による TypeError: Ut[e] is not a constructor を回避するため、各ライブラリを固定バージョンで読み込み、直後に退避し、グローバル名を消去して安全に切替できるようにしています。

エラー修正の要点

  • Quagga と Quagga2 は同じグローバル名 Quagga を用いるビルドがあるため、読み込み直後に別名へ退避して衝突を回避。
  • 固定バージョン（Quagga 0.12.1 / Quagga2 1.7.2 / ZXing 0.20.0）を使用してミニファイ差異の影響を排除。
  • アダプタは退避した参照（__Quagga_v0121__ / __Quagga2_v172__）のみを使用。

ZXing 高速化の要点

  • DecodeHintType.POSSIBLE_FORMATS = [ EAN_13 ] に限定。
  • 解像度を 720p などに制限して総ピクセル数を抑制。
  • 最初のヒットで停止（連続処理を避ける）。
